# LME Model Documentation

## Overview

This document describes the Linear Mixed Effects (LME) model used in the CTG stability analysis.

## Input Files

The LME analysis uses the CTG results generated by the pipeline:

- **Input:** `{cache_dir}/results/{eid}_ctg_results.npz`
- **Format:** NumPy compressed archive containing:
  - `ctg_mean`: Mean cross-temporal generalization matrix
  - `ctg_std`: Standard deviation of CTG matrix
  - `self_consistency_mean`: Mean self-consistency over time
  - `self_consistency_std`: Standard deviation of self-consistency
  - `n_neurons`: Number of neurons
  - `n_trials`: Number of trials

## Analysis Pipeline

1. Load results from multiple sessions
2. Extract stability metrics from CTG matrices
3. Fit LME model with session as random effect
4. Analyze temporal dynamics

## Configuration

Results are generated using parameters from `config.yaml`:

```yaml
analysis:
  bin_size: 0.05              # Time bin resolution
  window_sec: [-0.2, 0.8]     # Analysis window
  decoder_window_bins: 3      # Decoder window size
  n_folds: 5                  # Cross-validation folds
  n_iterations: 30            # Bootstrap iterations
```
